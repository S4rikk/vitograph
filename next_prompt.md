# SURGICAL REPAIR: FIXING NUTRIENTS IN CARDS

Проблема найдена: парсер в `food-log-parser.ts` (строка 49) слишком хрупкий. Он не видит значения нутриентов, если между числом и единицей измерения есть пробел (например, `15 мкг`), а ИИ часто добавляет этот пробел для красоты.

## ТВОЯ ЗАДАЧА (СТРОГО ОГРАНИЧЕННАЯ):

### 1. Исправь `apps/web/src/components/diary/food-log-parser.ts`
Обнови только логику извлечения значений (линия 49), сделав регулярное выражение более надежным:

```typescript
// БЫЛО:
// const valueMatch = content.match(/([\d.,]+[A-Za-zА-Яа-я]+)[\s)]*$/);

// СТАЛО (добавлен \s* для поддержки пробелов и %):

```

**Больше ничего не трогай.** Не меняй промпты, не меняй логику в `tools.ts`. Только это маленькое, но критическое изменение в парсере.
