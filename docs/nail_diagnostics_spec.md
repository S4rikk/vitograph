# VITOGRAPH: Nail Diagnostics (Visual Biomarkers)

## 1. Концепция и Роль в Системе
Модуль "Диагностика по ногтям" (Nail Diagnostics) представляет собой первую интеграцию компьютерного зрения (Computer Vision) в VITOGRAPH. 
Так как ногти растут медленно (около 3 мм в месяц), их состояние является **долгосрочным архивом метаболических процессов и дефицитов** (за последние 3-6 месяцев).

Этот модуль дополняет Опросник (50 маркеров) и Анализы крови, позволяя выявлять латентные (скрытые) дефициты еще до того, как они критически отразятся в крови. Платформа будет анализировать фотографию рук пользователя и сопоставлять визуальные маркеры с базой `biomarkers`.

## 2. Основные визуальные маркеры ногтей (Nail Biomarkers)

| Визуальный симптом | Медицинский термин | Потенциальные дефициты / Состояния (Ключи для LLM/CV) |
| :--- | :--- | :--- |
| **Белые пятна/полоски** | Лейконихия (Leukonychia) | Дефицит **Цинка**, дефицит кальция, избыток углеводов. |
| **Продольные борозды** | Longitudinal striations | Снижение усвоения белка, дефицит **Железа** / B12 (анемия), возрастное снижение кислотности желудка. |
| **Поперечные борозды** | Линии Бо (Beau's lines) | Перенесенный сильный стресс, инфекция (цинк/белок ушли на иммунный ответ), дефицит цинка. |
| **Ломкость, расслаивание** | Онихоррексис (Onychorrhexis) | Дефицит коллагена/белка, дефицит кремния, кальция, **Гипотиреоз** (щитовидная железа). |
| **Бледные (белые) ногти / ложе** | Terry's nails / Pale nail bed | **Анемия** (железо), застой в печени, дефицит кровообращения. |
| **Ложкообразные ногти** | Койлонихия (Koilonychia) | Тяжелая **Железодефицитная 아немия** (ЖДА), гемохроматоз. |
| **Желтые ногти** | Yellow nail syndrome | Грибковая инфекция, проблема с лимфотоком, избыток сахара (гликация). |
| **Отсутствие лунок (Lunula)** | Absent lunula | Дефицит витамина **B12**, истощение нервной системы (надпочечники), снижение работы щитовидки. |
| **Красные лунки** | Red lunula | Сердечно-сосудистые проблемы, системное воспаление. |
| **Утолщенные ногти "часовые стекла"** | Clubbing (Hippocratic fingers) | Хроническая гипоксия (нехватка кислорода), проблемы с легкими/сердцем. |
| **Точечные углубления (наперсток)** | Pitting | Псориаз, аутоиммунные процессы, дефицит витамина D. |

## 3. Интеграция с Архитектурой (MVP)

Для MVP (Phase 27) нам не обязательно сразу обучать собственную модель компьютерного зрения. Мы можем использовать мощь мультимодальных LLM (например, `gpt-4o` или `claude-3-5-sonnet`), передавая им фото ногтей пользователя с жестким системным промптом, настроенным на поиск вышеописанных маркеров.

### Предлагаемый Workflow:
1.  **UI (Frontend):** В разделе "Анализы" (или отдельным виджетом на главной) добавить карточку: `Сфотографируй свои ногти без лака — узнай скрытые дефициты`.
2.  **API (Node.js/LangGraph):** Фотография загружается и отправляется на `Vision Node` в LangGraph.
3.  **Process (Vision LLM):** LLM анализирует изображение на основе нашего `Nail Biomarkers Prompt`. Важно: LLM должна выдать *вероятные* маркеры (JSON), а не ставить медицинский диагноз.
4.  **Integration (Dynamic Norms):** Выявленные маркеры сохраняются в `profiles.lifestyle_markers` под ключом `somatic_data.nails`.
5.  **Output (AI Assistant):** AI-друг пишет пользователю: *"Я проанализировал фото. Вижу белые пятнышки и продольные борозды. Зачастую это говорит о том, что у организма повышенный расход цинка и железа. С учетом того, что у тебя по опроснику сильный стресс, это ожидаемо! Давай добавим в рацион больше тыквенных семечек."*

## 4. Следующие шаги
